# MAML_CP_VAE
This repo contains Pytorch implementation of the paper ST2:Small-data Text Style Transfer via Multi-task Meta-Learning. Up to now only CP-VAE version is contained, CrossAlign and VAE versions will be 
updated soom.

## Dependencies
- torch==1.3.1
- python > 3.6
- tqdm
- pandas
- numpy
- scipy
